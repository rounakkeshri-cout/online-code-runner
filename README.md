# 💻 Online Code Runner

🚀 **Live Demo:** https://frontened-project.vercel.app/

A simple and interactive **Online Code Runner** that allows users to write, edit, and execute code directly in the browser.
This project is designed to simulate a lightweight coding environment for frontend development practice.


## ✨ Features

* 🧑‍💻 Write and edit code in browser
* ⚡ Instant output execution
* 🎯 Clean and simple UI
* 📱 Responsive design
* 🌐 No installation required
* 🚀 Deployed on Vercel



## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Deployment

* Vercel



## 📂 Project Structure

```
online-code-runner/
│── index.html     # Main UI
│── style.css      # Styling
│── script.js      # Logic for code execution
```



## 🚀 How It Works

1. User writes code in the editor
2. Clicks the **Run** button
3. JavaScript processes the code
4. Output is displayed instantly
